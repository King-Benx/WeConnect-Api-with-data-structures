WeConnect API

WeConnect provides a platform that brings businesses and individuals together. This platform creates awareness for businesses and gives the users the ability to write reviews about the businesses they have interacted with.

Access Our API Through The Following Routes

Functionality Method Parameters Routes
Creates a user account POST username, password, email {{ context.register_user }}
Login User POST username, password {{ context.login }}
Logs out a user POST username {{ context.logout_user }}
Password Reset POST username, new_password {{ context.reset_password }}
Register a business POST name, location, category, description {{ context.register_business }}
Update a business PUT businessId, name, location, category,description {{ context.update_business }}
Retrieve all Businesses GET {{ context.retrieve_all_businesses }}
Remove a business DELETE businessId {{ context.delete_business }}
Get a Business GET businessId {{ context.retrieve_a_business }}
Add a Review for business POST review {{ context.review_a_business }}
Get all Reviews for business GET businessId {{ context.review_a_business }}
